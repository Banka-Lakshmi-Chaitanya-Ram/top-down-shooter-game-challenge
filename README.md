# top-down-shooter (game challenge [ GAME ] )

## Overview

This is a Unity-based top-down shooter project.

When you download and open this project, Unity will automatically generate some folders (like Library, Temp, etc.).
👉 No need to worry about these — they are created by Unity itself.

---

## Unity Version

This project can be opened in most Unity versions.
Even if you use a newer version, Unity will automatically upgrade the project if needed.

---

## Opening the Project

After opening the project in Unity:

* You may see a **blank screen initially** — this is normal.
* To load the game scene:

### Steps:

1. Go to **Assets**
2. Open the **Scenes** folder
3. Double-click on **SampleScene**

👉 This will load the main game scene.

---

## Accessing Project Files

All scripts, assets, and game data are available inside the **Assets** folder.

---

## Build & Run the Game

To build and run the game on your device:

### Steps:

1. Open Unity
2. Go to **File → Build Settings**
3. Select your platform:

   * Windows (PC)
   * macOS
4. Click **Build**
5. Choose a location to save the build

After building, you can run the game on your device.

---

## Compatibility

* Works on **Windows and macOS**
* Compatible with most modern systems (including macOS with Apple Silicon)

---

## Notes

* Auto-generated folders (Library, Temp, etc.) are intentionally excluded.
* Unity will recreate them when the project is opened.
